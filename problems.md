| 任务             | 验收标准                                  |
| -------------- | ------------------------------------- |
| 后端真实 abort 一致性 | 用户停止后，数据库最终状态是 `ABORTED`              |
| 错误状态一致性        | 模型/API 报错后是 `FAILED`，不能残留 `STREAMING` |
| 多会话防串线         | A 会话生成时切到 B，会话内容不串                    |
| 回归测试           | 至少覆盖 `done / error / aborted` 三条路径    |
