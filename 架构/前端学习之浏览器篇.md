# 💻 前端必备浏览器核心知识体系

作为一名进阶前端工程师，对浏览器的理解是架构和性能优化的基础。本学习规范围绕**运行时、性能和安全**三大核心维度展开。

## 阶段一：浏览器渲染与运行时 (The Runtime)

这部分是理解代码如何转化为像素、以及如何影响组件更新的基础。

| 知识点 | 核心内容 | 推荐学习资料 |
| :--- | :--- | :--- |
| **关键渲染路径 (CRP)** | 理解从 HTML/CSS/JS 接收到屏幕上像素绘制的完整流程：**DOM Tree, CSSOM Tree, Render Tree, Layout, Paint, Compositing**。| **文章:** "Critical Rendering Path" (Google Developers - 经典系列) <br/>**搜索关键词:** `Critical Rendering Path MDN` |
| **回流与重绘 (Reflow & Repaint)** | 理解哪些 CSS/JS 操作会触发 Layout (回流)，哪些只触发 Paint (重绘)。优化目标：**尽量只触发 Compositing**。 | **文章:** "高性能 CSS 与 JavaScript 动画" (Google Developers) <br/>**搜索关键词:** `回流重绘 优化` |
| **浏览器存储机制** | **Cookie, LocalStorage, SessionStorage, IndexedDB** 的区别、容量、生命周期、安全性（`HttpOnly`）。| **MDN 文档:** Web Storage API <br/>**搜索关键词:** `Web Storage MDN` |
| **浏览器事件循环 (Event Loop)** | 深入理解 V8 引擎中的 **Call Stack, Web APIs, Event Queue**。区分 **Microtasks (微任务)** 和 **Macrotasks (宏任务)** 的执行顺序。| **视频:** "In The Loop" - Jake Archibald (YouTube) - *经典讲解* <br/>**搜索关键词:** `Jake Archibald event loop` |

## 阶段二：性能优化与网络 (Performance & Network)

这部分是提高应用响应速度和用户体验的根本。

| 知识点 | 核心内容 | 推荐学习资料 |
| :--- | :--- | :--- |
| **网络请求流程** | 理解从输入 URL 到页面显示的完整过程：**DNS 解析、TCP 握手、TLS 握手、HTTP 请求与响应**。| **文章:** "从输入 URL 到页面渲染发生了什么" (经典面试题) <br/>**搜索关键词:** `输入 URL 发生什么 知乎` |
| **HTTP 协议进阶** | **HTTP/1.1, HTTP/2, HTTP/3 (QUIC)** 的对比。理解 HTTP/2 的多路复用、服务器推送。掌握 HTTP 缓存（`Cache-Control`, `ETag`, `Last-Modified`）。| **MDN 文档:** HTTP Caching <br/>**视频:** "HTTP/3 Explained" <br/>**搜索关键词:** `HTTP 缓存 MDN` |
| **资源加载优化** | **Preload, Preconnect, Prefetch** 等资源预加载指令的使用场景和效果。**图片优化**（WebP, AVIF, Lazy Loading, Srcset）。| **文章:** "Preload, Prefetch and Priorities in Chrome" (Google Developers) <br/>**搜索关键词:** `preload prefetch chrome` |
| **渲染性能工具** | 熟练使用 Chrome DevTools 的 **Performance (性能) 面板**分析帧率、主线程活动和内存泄露。| **视频/文档:** Chrome DevTools 官方文档 - Performance Panel <br/>**搜索关键词:** `Chrome DevTools Performance 教程` |

## 阶段三：安全与高级特性 (Security & Advanced)

这部分是保证应用健壮性、防止跨站攻击的关键。

| 知识点 | 核心内容 | 推荐学习资料 |
| :--- | :--- | :--- |
| **浏览器安全** | 深入理解 **XSS (跨站脚本攻击)** 和 **CSRF (跨站请求伪造)** 的原理和防范措施。掌握 **CORS (跨域资源共享)** 的预检请求流程。 | **文章:** OWASP Top 10（了解主要威胁）<br/>**搜索关键词:** `XSS CSRF 原理 防范` |
| **同源策略 (SOP)** | 理解同源策略对 Cookie、DOM 和 XHR 请求的限制，以及如何通过 JSONP、CORS 和代理解决跨域问题。| **MDN 文档:** Same-origin policy <br/>**搜索关键词:** `同源策略 MDN` |
| **Web Workers** | 理解主线程和 Worker 线程的区别，以及如何使用 Worker 处理复杂的计算任务，避免阻塞主线程。| **MDN 文档:** Using Web Workers <br/>**搜索关键词:** `Web Workers MDN` |
| **新的 API 趋势** | 了解 **Web Component, WebGL, WebAssembly (Wasm)** 的基础概念和应用场景，作为未来的技术储备。| **MDN 文档:** WebAssembly 概念 <br/>**搜索关键词:** `WebAssembly 入门` |
