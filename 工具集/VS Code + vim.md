### tips
#### 1.想查看或者跳转到某个函数的实现
gd =>跳转到定义(go to definition)
ctrl + o => 返回
#### 2.想看某个变量的提示词
gh => 鼠标移动后显示(go + hover）

#### 3.跳转到编辑器左右两边的文件
gt/gT
1gt => 跳转到第一个
2gt => 跳转到第二个
#### 4. 跳转到侧边栏
command + 0
选中文件后按 space => 打开文件
编辑打开后的文件 => o
#### 5.代码左右分屏左右切换
command +1/2
